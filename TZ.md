# 1. Общая информация

## Цель проекта

Разработать современный веб-сервис для прогнозирования результатов этапов чемпионата Formula 1 сезона 2026.

Пользователь должен иметь возможность:

- просматривать календарь сезона;
- просматривать результаты уже завершённых этапов;
- делать прогнозы на квалификацию;
- делать прогнозы на гонку;
- просматривать свои прогнозы;
- видеть рейтинг пользователей.

Интерфейс полностью на украинском языке.

Сайт должен корректно работать на:

- ПК;
- планшетах;
- мобильных устройствах.

# 6. Страница квалификации

Выбор режима

TOP-10

Если выбран Подиум

Показать три позиции

P1

P3

Если выбран TOP-10

Показать

P1

...

P10

Каждая позиция представляет собой пустой слот.

При нажатии открывается окно выбора пилота.

# 7. Выбор пилота

Открывается модальное окно.

Для каждого пилота отображается:

Фотография

Имя

Фамилия

Трёхбуквенный код

Логотип команды

После выбора карточка автоматически занимает выбранную позицию.

Один пилот не может быть выбран дважды.
